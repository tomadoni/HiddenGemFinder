# Soccer Hidden Gem Finder

This project identifies undervalued attacking players across the Premier League and La Liga using advanced soccer analytics.

## Features

The model evaluates players using:

- Goals and Assists
- Expected Goals (xG)
- Expected Assists (xA)
- Per-90 production
- Age
- Market value from Transfermarkt

Players are ranked using a **Hidden Gem Score** that rewards:

- Strong underlying attacking metrics
- Younger players
- Lower market value relative to performance

## Tech Stack

Python  
Pandas  
Scikit-learn  
Streamlit  
Plotly

## Data Sources

- FBref (player performance statistics)
- Transfermarkt (player market values)
